# PRD: 닌자거북의홈 (HOMEninja)

## 목표
인디 앱 개발자의 모든 앱을 한 페이지에서 소개하고 다운로드할 수 있는 포트폴리오 사이트.
커뮤니티/게시판에 이 하나의 링크만 공유하면, 방문자가 앱을 발견하고 즉시 다운로드한다.

## 사용자
- **주 사용자**: 커뮤니티/게시판(클리앙, 뽐뿌, Reddit 등)에서 링크를 클릭해 들어오는 잠재 사용자
- **유입 경로**: 게시판 글 → 링크 클릭 → 홈페이지 도착
- **디바이스**: 모바일 70% 이상 (게시판 앱/모바일 브라우저에서 유입)
- **체류 시간**: 짧음 (30초~1분). 빠르게 훑고 다운로드하거나 이탈

## 핵심 컨셉: 멀티 은하 시스템

사이트는 3개의 "은하"로 구성되며, 방향키(← →) 또는 하단 네비게이션으로 전환한다.

| 은하 | 내용 | 오브젝트 |
|------|------|----------|
| 태양계 (Solar System) | 출시된 앱 | 행성 (Planet) |
| 운석 지대 (Asteroid Field) | 개발 중/아이디어 | 운석 (Asteroid) |
| 빛나는 성운 (Bright Nebula) | 미래 공간 | 성운 오브 (Nebula) |

## 기능

### F-001: 히어로 (첫 은하에서만)
- NinjaTurtle 로고 + 타이틀
- 한 줄 소개 ("앱으로 세상을 조금 더 편하게")
- 은하 탐색 안내 ("← → 방향키나 아래 버튼으로 다른 은하 탐색")
- 지구 버튼 (Earth) — 후원 페이지 진입점

### F-002: 은하 네비게이션
- 하단 고정 네비게이션 바 (← ● ● ● →)
- 방향키(← →)로 전환
- 하이퍼스페이스 전환 이펙트
- 마지막 방문 은하 localStorage 저장/복원

### F-003: 행성 섹션 (태양계)
- 앱 이름, 설명, 부가정보 (OS 버전, 업데이트 날짜)
- CSS gradient 행성 그래픽 (ring 지원)
- 플랫폼 감지 다운로드 버튼
- 렌즈플레어 이펙트 (랜덤 위치/각도/줄기 수)

### F-004: 운석 섹션 (운석 지대)
- 개발 중/아이디어 앱을 운석으로 표현
- 클릭 시 모달로 상세 확인
- 황야 톤 배경 (dustHaze + amber star tint)

### F-005: 성운 섹션 (빛나는 성운)
- 미래 앱 공간을 성운 오브로 표현
- 보라 톤 가스 밴드 배경 (gasBands + purple star tint)

### F-006: 별 배경 + 유성
- Canvas 별 배경 (은하별 밀도/색조 차별)
- 랜덤 유성 (방향, 속도, 꼬리 길이, 스폰 간격 모두 랜덤)

### F-007: 후원 (Earth → EarthZoom)
- 지구 버튼 클릭 → 6단계 줌인 오버레이
  - 지구 → 아시아 → 한반도 → 대한민국 → 제주도 → 우리집
- 마지막 단계: 닌자거북이 작업실 카드 + 토스 QR 코드
- ESC/닫기로 복귀, 방향키/dots로 단계 이동

### F-008: 개인정보처리방침
- `/privacy` — 통합 페이지
- `/privacy/callninja` — CallNinja 전용
- `/privacy/spamcall070` — SpamCall070 전용
